# [AUTH] 统一路径前缀联调（/api/v1）

- 模块: AUTH
- 责任人: FE: @xxx / BE: @yyy
- 关联任务: AUTH-002

## 背景
- 前端接口使用 '/auth/*'
- 后端实际为 '/api/v1/auth/*'

## 目标
- 统一前后端路径前缀，避免 404/路由不一致

## 方案
- 方案A（推荐）: Axios baseURL 设为 '/api/v1'
- 方案B: 各模块 API 手动补 '/api/v1' 前缀

## 验收
- 登录/注册/登出/刷新/me/发送验证码/重置密码 全部 200
- 前端无需额外拼接 '/api/v1'

## 风险/依赖
- 反向代理/网关现有规则（若有）
